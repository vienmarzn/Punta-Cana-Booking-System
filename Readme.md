#How To RUN PROJECT:

How to run the Punta Cana Booking System using PHP and MySQL

1. Download the project zip file

2. Extract the file and copy codeme_db.sql folder

3. Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/Html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with the name  codeme_db

6. Import codeme_db.sql file(given inside the zip package in SQL file folder)

7. Run the script http://localhost/bbs
